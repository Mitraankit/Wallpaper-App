# Wallpaper App (React + Vite)

Mobile-friendly wallpaper browser using the Wallhaven API:

- Search + filters (categories, sorting, top range, min resolution, ratios, color)
- Infinite scroll
- Fullscreen preview modal with tags + download + share + copy link
- Favorites stored locally (no login)

## Run

```bash
cd /Users/ankitkumar/coding/wallpaper-app
npm install
npm run dev
```

## Build

```bash
npm run build
```

The Vite `base` is set to `./` in `vite.config.js` so the build works on GitHub Pages and other static hosts.

## Notes

- Wallhaven may restrict NSFW/sketchy results without an account; the app defaults to SFW.
